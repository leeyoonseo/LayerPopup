_createElement(tag, id, className, name, type, label, src, text) → {object}

객체 생성하는 함수, 버튼 생성할때는 호출 시 .call(this) 바인딩해줘야 함.

Example
title = _createElement({ 
     tag : 'p', 
     className : 'title,title-red,title-required' 
});
     
button = _createElement.call(this,{ 
     tag : 'button', 
     className : 'cancel',
     type : 'button', 
     label : '취소버튼' 
});
