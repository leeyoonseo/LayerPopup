만료일

만료일-기본형

Parameters :
expire Boolean 만료일 사용 여부
expireData 설정하지 않는다면 기본 문구와 날짜가 설정된다.
expireData Object or Array 만료일 날짜, 문구 설정
expire가 false이면 사용 불가능
this.FoxLayerPopup.close(true or false) 커스텀 event시 호출 true일 경우 만료일 저장
Source Code :

const popup = new LayerPopup({
    expire : true
});
        
element.addEventListener('click', () => {
    popup.open();
});

만료일-단수

Parameters :
expire Boolean 만료일 사용 여부
expireData 설정하지 않는다면 기본 문구와 날짜가 설정된다.
expireData Object or Array 만료일 날짜, 문구 설정
expire가 false이면 사용 불가능
this.FoxLayerPopup.close(true or false) 커스텀 event시 호출 true일 경우 만료일 저장

const popup = new LayerPopup({

만료일-커스텀

Parameters :
expire Boolean 만료일 사용 여부
expireData 설정하지 않는다면 기본 문구와 날짜가 설정된다.
expireData Object or Array 만료일 날짜, 문구 설정
expire가 false이면 사용 불가능
this.FoxLayerPopup.close(true or false) 커스텀 event시 호출 true일 경우 만료일 저장
Source Code :

const popup = new LayerPopup({
    customButton : true,
    customButtonData : [
        {
            type : 'button',
            className : 'custom_done',
            label : '커스텀 확인',
            event : function(){
                this.LayerPopup.close(true);
            }
        },
        {
            type : 'button',
            className : 'custom_cancel',
            label : '커스텀 취소',
            event : function(){
                this.LayerPopup.close(false);
            }
        }
    ],
    expire : true,
    expireData : [
        {
            date : 1,
            id : 'custom_day',
            label : '하루동안보지않기'
        },
        {
            date : 7,
            id : 'custom_week',
            label : '일주일동안보지않기'
        }
    ]
});

element.addEventListener('click', () => {
    popup.open();
});

버튼-이벤트 설정

Parameters :
event : function(){} Function 사용자설정 버튼 클릭 시 이벤트
this.FoxLayerPopup.close()로 팝업닫기 가능
Source Code :

const popup = new LayerPopup({
    customButton : true,
    customButtonData : [
        {
            type : 'button',
            label : '메세지변경',
            event : function(){
                const {content} = this.LayerPopup;
                content.innerText = '메세지를 변경했습니다!.'
            }
        },
        {
            type : 'button',
            label : '만료일팝업열기',
            event : function(){
                expirePopup.open();
            }

        },
        {
            type : 'button',
            label : '닫기'
        }
    ]
});
        
element.addEventListener('click', () => {
    popup.open();
});

콜백함수

Parameters :
callback Function 기본 이미지 버튼 클릭 시 callback함수 설정
Source Code :

const popup = new LayerPopup(function(res){
    const message = (res) 
        ? '확인을 클릭하셨습니다.' 
        : '취소를 클릭하셨습니다.';

        alert(message);
});
        
element.addEventListener('click', () => {
    popup.open();
});

여러개의 팝업 호출

Source Code :

const firstPopup = new LayerPopup({
    title : '첫번째 팝업',
    content : '2초 후 2번째 팝업이 오픈됩니다.'
});
const secondPopup = new LayerPopup({
    title : '두번째 팝업',
    content : '버튼 클릭 시 3번째 팝업이 오픈됩니다.',
    customButton : true,
    customButtonData : {
        label : '새로운 팝업열기',
        event : function(){
            thirdPopup.open();
        }
    }
});
const thirdPopup = new LayerPopup({
    title : '세번째 팝업',
    content : '팝업을 전체 다 닫아야지 배경이 제거됩니다.'
});
document.getElementById('overlap').addEventListener('click', () => {
    firstPopup.open();

    setTimeout(() => {
        secondPopup.open();
    },2000);
});

    const customData = {
        title : '<p class="custom_sale_sub_title">Christmas</p>'
                + '<p class="custom_sale_main_title">BIG<br>SALE</p>',

    }

        className : 'custom_sale',

        content : customData.content,
        expire : true,

            data : 1,
